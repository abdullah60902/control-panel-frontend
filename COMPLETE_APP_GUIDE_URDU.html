System Kya Hai?

Yeh ek Care Home Management System hai jo care homes ko digitally manage karne ke liye banaya gaya hai. Isme residents, staff, medications, compliance, aur daily operations sab kuch track hota hai.

üéØ Main Purpose: Paper-based records ko replace karke sab kuch digital, safe, aur accessible banana.

üë• Part 3: Resident Management

Resident Profile Kaise Banayein?

  1. Dashboard par + Add Resident button click karein
  2. Basic details bharein: Name, DOB, Room Number, Admission Date
  3. Save click karein
  4. Profile ban jayegi aur aap additional details add kar sakte hain

Resident Profile Ke Sections:

1. About Me (Personal Info)

  • Personal Details: Name, age, gender, contact info
  • Emergency Contacts: Family members ke numbers
  • Medical History: Allergies, chronic conditions
  • Preferences: Food likes/dislikes, hobbies

Buttons:

  • Edit - Information update karne ke liye
  • Save - Changes save karne ke liye

2. Care Plan

  • Daily Care Needs: Bathing, dressing, mobility support
  • Care Goals: "Improve walking", "Reduce anxiety"
  • Review Dates: Kab next review hoga

Buttons:

  • Add Care Plan - Naya plan banayein
  • Update Plan - Existing plan edit karein
  • Print PDF - Care plan print karein

3. Medication (eMAR)

  • Medicine List: Sari medicines ka naam, dosage, timing
  • MAR Chart: Kon si medicine kab di gayi
  • Stock Levels: Kitni tablets/bottles remaining hain

Buttons:

  • GIVE - Medicine di gayi (green button)
  • REFUSE - Resident ne refuse kiya (red button, reason mandatory)
  • Add Medicine - Nayi medicine add karein
  • View History - Past medication records

4. Daily Log

  • Daily Notes: Staff ke observations
  • Food/Fluid Intake: Kitna khaya/piya
  • Mood/Behavior: Resident ka mood kaisa tha
  • Activities: Kya activities kiye

Buttons:

  • Add Entry - Naya log entry
  • Filter by Date - Specific date ke logs

5. Risk Assessment

  • Fall Risk: Girne ka khatra kitna hai
  • Choking Risk: Khane mein problem
  • Skin Integrity: Pressure sores ka risk
  • Mental Capacity: Decision-making ability

Buttons:

  • Assess Risk - Naya assessment
  • Update - Existing assessment update

6. Documents

  • Upload Documents: Scanned papers, certificates
  • Categories: Medical reports, consent forms, photos
  • View/Download: Documents ko dekh aur download kar sakte hain

Buttons:

  • Upload - Nayi file upload karein
  • View - Document open karein
  • Delete - Document remove karein (admin only)

7. Handover Notes

  • Shift Handover: Ek shift se dusri shift ko information
  • Important Updates: "Resident unwell", "Family visiting"

Buttons:

  • Add Handover - Naya note likhein
  • Mark as Read - Note padh liya

8. PBS Plan (Positive Behavior Support)

  • Behavior Triggers: Kya cheez se resident upset hota hai
  • De-escalation: Kaise calm karein
  • Strategies: Behavior manage karne ke tareeqe

Buttons:

  • Create Plan - Naya PBS plan
  • Update - Plan modify karein

9. Goals & Outcomes

  • Set Goals: "Walk 10 steps daily"
  • Track Progress: Goal achieve ho raha hai ya nahi
  • Outcomes: Results record karein

Buttons:

  • Add Goal - Naya goal set karein
  • Update Progress - Progress record karein

10. Consent Forms

  • Medical Consent: Treatment ke liye permission
  • Photo Consent: Photos lene ki permission
  • Activity Consent: Outings ke liye consent

Buttons:

  • Upload Consent - Signed form upload karein
  • View - Consent form dekhein

üíä Part 4: Medication Module (eMAR)

eMAR Kya Hai?

Electronic Medication Administration Record - Digital MAR chart jo paper charts replace karta hai.

Medication Round Kaise Karein?

  1. Sidebar se Medication section open karein
  2. Time slot select karein: Morning (8am), Afternoon (2pm), Evening (6pm), Night (10pm)
  3. Har resident ke liye medicines list dikhegi
  4. Medicine dene ke baad GIVE button click karein
  5. Agar resident refuse kare to REFUSE click karein aur reason likhen

Medication Screen Ke Buttons:

Button Color Kya Hota Hai
GIVE Green Medicine di gayi. Stock automatically reduce hota hai. Timestamp record hota hai.
REFUSE Red Resident ne medicine refuse ki. Reason mandatory hai (e.g., "Sleeping", "Nausea")
Add Medicine Blue Naya prescription add karein. Name, dosage, frequency, start date bharein.
Edit Yellow Existing medicine ki details change karein (dosage, timing)
Discontinue Red Medicine band karni hai (doctor's order chahiye)
Stock Alert Orange Agar stock kam hai to alert dikhata hai. Reorder karne ka reminder.
Print MAR Gray Monthly MAR chart PDF mein download karein
‚ö†Ô∏è Important: Agar medicine PRN (as needed) hai, to har baar dene se pehle reason likhna mandatory hai.

üìã Part 7: Incident Reports

Incident Report Kab Banayein?

  • Resident gir gaya (Fall)
  • Injury hui (Cut, bruise, etc.)
  • Medication error (Wrong dose, missed dose)
  • Behavioral incident (Aggression, wandering)
  • Equipment failure (Hoist broke, bed malfunction)
  • Safeguarding concern (Abuse suspicion)

Incident Report Kaise Banayein?

  1. Dashboard ya sidebar se Incident Reports open karein
  2. + New Incident click karein
  3. Form bharein:
    • Incident Type select karein
    • Date & Time
    • Location (Room number, bathroom, etc.)
    • Description (Kya hua detail mein)
    • Witnesses (Kon present tha)
    • Action Taken (Kya kiya - first aid, doctor call, etc.)
  4. Submit click karein
  5. Manager ko automatically notification jayegi

Incident Report Ke Buttons:

Button Function
New Incident Naya incident report create karein
View Details Existing incident ki full details dekhein
Add Follow-up Incident ke baad kya action liya, update karein
Upload Photo Injury ya damage ki photo upload karein (evidence)
Mark Reviewed Manager ne review kar liya (approval)
Export PDF Incident report print karein (insurance/legal)
6-Month Overview Chart dikhata hai - kitne incidents hue, trends kya hain
‚úì Best Practice: Incident report 24 hours ke andar complete karein. Family ko inform karein agar serious incident hai.

üéØ Summary: System Kaise Kaam Karta Hai

Daily Workflow Example:

  1. Morning (8am):
    • Login karein
    • Dashboard check karein - koi alerts?
    • Medication round karein - har resident ko medicines dein, GIVE button click karein
    • Daily logs mein notes likhein - "Mrs. Smith slept well"
  2. Afternoon (2pm):
    • Lunch medication round
    • Agar koi incident ho (fall, etc.) to incident report banayein
    • Care plans review karein
  3. Evening (6pm):
    • Evening medication round
    • Handover notes likhein next shift ke liye
    • Compliance checks complete karein (water temp, fridge temp)
  4. Night (10pm):
    • Night medication round
    • Final checks
    • Logout

Key Points Yaad Rakhein:

  • Green Buttons = Positive actions (Save, Give, Complete)
  • Red Buttons = Negative/Delete actions (Refuse, Delete, Discontinue)
  • Blue Buttons = Add/Create new records
  • Yellow Buttons = Edit/Update existing records
  • Red Alerts = Urgent action required
  • Yellow Alerts = Warning, jaldi address karein

  2. Us button par click karein

  4. "Save as PDF" option select karein
  5. File name dein aur Save click karein
  6. PDF download ho jayegi!
