## üìã Care Plan Types Guide

### 1Ô∏è‚É£ Personal Hygiene Care Plan

**When to Use:**
- Resident needs assistance with washing, bathing, or showering
- Help required with dressing/undressing
- Grooming support needed (hair, nails, shaving)
- Skin care or pressure area management

**Key Fields:**
- Washing/Bathing Instructions
- Dressing Instructions
- Grooming Instructions
- Skin Care Protocol
- Products & COSHH Notes

**Example:**
```
Current Ability: Can wash hands independently
Care Aims: Maintain skin integrity
Washing: Assist with shower MWF at 9am
Dressing: Full assistance with socks and shoes
Grooming: Electric shaver only, supervised
Skin Care: Check heels and coccyx daily
```

---

### 2Ô∏è‚É£ Moving and Handling Care Plan

**When to Use:**
- Resident has mobility limitations
- Transfer assistance required
- Fall risk present
- Equipment needed for safe movement

**Key Fields:**
- Current Mobility Level
- Equipment Required
- Transfer Techniques
- Number of Staff Required
- Risk Assessment Notes

**Example:**
```
Mobility: Walks short distances with frame
Equipment: Walking frame, wheelchair for longer trips
Transfers: 2-person assist, use slide sheet
Staff: Minimum 2 for all transfers
Risk: High fall risk, never leave unattended
```

### 3Ô∏è‚É£ Nutrition and Hydration Plan

## üéØ Best Practices

### Creating Quality Care Plans

**DO:**
- ‚úÖ Be specific and detailed
- ‚úÖ Use clear, professional language
- ‚úÖ Include measurable goals
- ‚úÖ Set realistic review dates
- ‚úÖ Upload supporting documents
- ‚úÖ Involve the resident in planning
- ‚úÖ Consider all aspects of care

**DON'T:**
- ‚ùå Use vague descriptions
- ‚ùå Skip important fields
- ‚ùå Set unrealistic goals
- ‚ùå Forget to set review dates
- ‚ùå Leave out important details
- ‚ùå Make assumptions without assessment

---

### Documentation Standards

**Language:**
- Use person-centered language
- Avoid medical jargon unless necessary
- Be objective and factual
- Use positive, respectful tone

**Details:**
- Include specific times and frequencies
- Document preferences and choices
- Note any changes from previous plans
- Record reasons for decisions

**Examples:**

**Good:**
```
"John prefers to shower in the morning at 8am. He can wash his upper body independently but requires assistance with his lower body and feet. Use the walk-in shower with shower chair. Water temperature should be warm, not hot."
```

**Poor:**
```
"Needs help with shower."
```

## üÜò Troubleshooting

### Common Issues

**Problem:** Can't find a resident's care plan
- **Solution:** Check filters are set to "All Plans" and "Filter All Settings"

**Problem:** Form won't submit
- **Solution:** Check all required fields are filled, especially dates

**Problem:** Attachment won't upload
- **Solution:** Check file size (<10MB) and format (supported types)

**Problem:** Can't edit care plan
- **Solution:** Verify you have edit permissions for this resident

**Problem:** Review alert not clearing
- **Solution:** Ensure you clicked "Mark as Reviewed" and set new review date
